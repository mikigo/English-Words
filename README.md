# English-Words

精心制作的 MarkDown 版免费学单词项目，包含各类英语词库，助您高效学习英语。

## 项目简介
这是一个开源的英语单词学习平台，提供了丰富的英语词汇资源，涵盖了各种考试和教材版本的词汇表。无论您是准备四六级、考研、雅思、托福，还是学习中小学英语教材，都能在这里找到合适的词汇资料。
